**Text From Image – OCR Desktop App**

A simple GUI-based application to extract text from images using Python, Tkinter, and EasyOCR.



**Features**

* Upload image via GUI (file dialog)
* Supports JPG, PNG, JPEG formats
* Extracts multi-language text using EasyOCR
* Displays output instantly
* Lightweight desktop tool – no API dependency



**Tech Stack**

* Python
* Tkinter (GUI)
* EasyOCR
* PIL (Image Processing)



**How to Run**

* pip install -r requirements.txt
* python app.py



**Future Improvements**
